# composer_install
instalador sh composer para distro linux V1.0

## Uso em linux

 - git clone **https://github.com/victorratts13/composer_instal.git**
 - cd composer_install
 - chmod +x install.sh
 - ./install.sh
 
 ## Uso em Docker
 
 - git clone **https://github.com/victorratts13/composer_instal.git**
 - cd composer_install
 - chmod +x install_docker.sh
 - ./install_docker.sh
